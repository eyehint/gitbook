# DELETE /device

### ThinQ 서버에서 기기 정보를 제거(unregister) 합니다.

#### Request

> **URL** : http://{device\_ip}:8951**/device**
>
> **METHOD** : <mark style="color:red;">**DELETE**</mark>
>
> **Header**&#x20;
>
> * message-id (option) : 로깅을 위한 id, 미 입력시 기기 내부에서 자동으로 생성
>
> **Parameter (Body)**
>
> * None

#### **Response**

> **Type** : Object

#### Response Example

```json
{}
```
