# DELETE /hub

### ThinQ 서버에 저장된 Hub를 제거(unregister) 합니다.

#### Request

> **URL** : http://localhost:8951**/hub**
>
> **METHOD** : <mark style="color:red;">**DELETE**</mark>
>
> **Header**&#x20;
>
> * message-id (option) : 로깅을 위한 id, 미 입력시 기기 내부에서 자동으로 생성
>
> **Parameter (Body)**
>
> * None

#### **Response**

> **Type** : Object

#### Response Example

```json
{}
```
